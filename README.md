# Titanic-data-cleaning-and-visualization
# Project Report

# Objective
To clean, analyze, and visualize the Titanic dataset using Python.

# Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

# Data Cleaning Performed
- Checked missing values
- Filled missing Age values
- Filled missing Embarked values
- Removed duplicate records

# Visualizations Created
- Survival Count
- Age Distribution
- Gender vs Survival
- Passenger Class vs Survival
- Correlation Heatmap

# Findings
1. Female passengers had a higher survival rate.
2. First-class passengers survived more often.
3. Most passengers were between 20 and 40 years old.
4. Missing values were handled successfully.

# Conclusion
The Titanic dataset was cleaned, analyzed, and visualized successfully using Python.
